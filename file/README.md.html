      1 # School Whistleblowing System
      2 
      3 A simple and privacy-focused web application built with Laravel 11 to help students safely report bullying incidents at school. This system empowers students to speak up without fear of exposure or retaliation.
      4 
      5 ## üéØ Purpose
      6 
      7 The goal of this project is to provide a secure, anonymous, and user-friendly platform for students who have witnessed or experienced bullying. It is designed to:
      8 
      9 - Protect the identity of whistleblowers
     10 - Encourage honest and safe reporting
     11 - Help school administrators handle reports effectively
     12 
     13 ## üîê Key Features
     14 
     15 - **Anonymous Reporting**: Users can choose to submit reports anonymously or include their identity if they're comfortable.
     16 - **Auto-generated Usernames**: During registration, usernames are randomly generated to protect user identity.
     17 - **Detailed Reporting**: Users can file reports specifying the incident, perpetrator information, and their role (victim/witness).
     18 - **User Dashboard**: Logged-in users can view, edit, and delete their own reports.
     19 - **Admin Panel**: Admins can view full report details, mark them as handled, and delete irrelevant or invalid reports.
     20 - **Role-based Access**: Only registered users can submit reports, and only admins can manage them.
     21 
     22 ## üõ† Tech Stack

     29 ## üöÄ Getting Started
     30 
     31 1. Clone the repo  
     32 
     33 ```bash
     34 git clone https://github.com/your-username/whistleblowing-system.git
     35 cd whistleblowing-system
     36 ```
     37 
     38 2. Install Dependencies
     39 ```bash
     40 composer install
     41 npm install && npm run build
     42 ```
     43 
     44 3. Set up your `.env`
     45 ```bash
     46 cp .env.example .env
     47 php artisan key:generate
     48 ```
     49 
     50 4. Set your DB credentials in `.env`, then run migrations:
     51 ```bash
     52 php artisan migrate
     53 ```
     54 
     55 5. Start the dev server
     56 ```bash
     57 npm install && npm run dev
     58 php artisan serve
     59 ```
